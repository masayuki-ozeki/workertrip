## 🔄 更新手順（マニュアル）

### 高知県に新しい施設を追加する場合

1. **js/data-kochi.js** を開く
2. `recommendedFacilities` 配列の末尾に新しいオブジェクトを追加:

```javascript
{
  name: "新しい施設名",
  type: "宿泊施設", // または "コワーキング"
  area: "エリア名",
  imageUrl: "https://example.com/image.jpg",
  description: "施設の説明（120文字程度）",
  features: ["Wi-Fi完備", "温泉", "会議室"],
  rakutenUrl: "https://hb.afl.rakuten.co.jp/...",
  jyaranUrl: "https://www.jalan.net/...",
  price: "¥10,000〜/泊",
  rating: 4.5
}
```

3. ファイルを保存
4. デプロイして完了

**注意事項**:
- 画像URLはUnsplashなどのCDN推奨（高速表示）
- リンクがない場合は空文字列（`""`）を設定
- 価格は分かりやすく「¥10,000〜/泊」形式で記載
- 評価は5段階（小数点1桁まで可）

## 🚀 他県への展開方法

### 既存の県に施設セクションを追加する場合

#### 1. データファイルに施設情報を追加
例: 東京都の場合、`js/data-tokyo.js` を編集

```javascript
// ファイル末尾に追加（module.exports の前）
const recommendedFacilities = [
  {
    name: "東京コワーキングスペース XXX",
    type: "コワーキング",
    area: "渋谷区",
    // ... 以下同様
  },
  // 最大10施設まで推奨
];
```

#### 2. HTMLファイルを更新
例: `tokyo/index.html` を編集

**Step A**: 「理由」セクションとCTAセクションの間に施設セクションを挿入

```html
<!-- おすすめ施設セクション -->
<section class="facilities-section">
    <div class="container">
        <div class="section-header">
            <span class="section-label">RECOMMENDED FACILITIES</span>
            <h2 class="section-title">東京のおすすめワーケーション施設</h2>
            <p class="section-description">
                東京都内のワーケーションに最適な施設をご紹介。
            </p>
        </div>
        <div class="facilities-grid" id="facilitiesGrid"></div>
    </div>
</section>
```

**Step B**: `</body>` タグの直前にJavaScriptコードを追加
（高知県のコードをそのままコピー可能）

#### 3. CSSは共通
`css/style.css` に既に施設セクション用のスタイルが含まれているため、追加不要

## 📝 使用方法

各県のindex.htmlファイルで以下のメタタグを更新：

```html
<meta property="og:image" content="{上記URL}">
```

**注意事項**:
- すべての画像はCreative Commons / パブリックドメインライセンス
- 画像URLは検索ツールが自動フィルタリング適用済み
- 直接URLを使用することでホスティングコスト削減
- 各画像は県の代表的ランドマークを表現

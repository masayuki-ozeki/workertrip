# WORKERTRIP – 全国ワーケーション情報ポータル

**はたらく旅が、はじまる。**

全国市区町村のワーケーション取り組みと補助金情報を集約したポータルサイトです。

## 🌟 プロジェクト概要

WORKERTRIPは、リモートワークの普及により可能になった「場所にとらわれない働き方」を支援するため、日本全国のワーケーション情報を一元管理するポータルサイトです。

### 現在の掲載状況

- **全国トップページ**: 47都道府県選択機能
- **長野県**: 48自治体の詳細情報掲載中（2026年度補助金2件）
- **山梨県**: 27自治体の詳細情報掲載中（富士吉田SHIGOTABIプロジェクト等）
- **新潟県**: 30自治体の詳細情報掲載中（西蒲区ワーケーションプラン支援事業あり）
- **その他都道府県**: 順次展開予定

### 主な特徴

- ✅ 全国市区町村のワーケーション取り組み情報
- ✅ 2026年度補助金制度の詳細
- ✅ 自治体公式ページへの直接リンク
- ✅ 検索・フィルター機能
- ✅ レスポンシブデザイン（全デバイス対応）

## 📝 データ管理

### 都道府県別データファイル

各都道府県の自治体データは個別ファイルで管理：

```javascript
// js/data-{prefecture}.js (例: data-nagano.js)
const municipalitiesData = [
    {
        name: "自治体名",
        url: "公式サイトURL",
        workationUrl: "ワーケーション情報URL",
        type: "dedicated" | "pdf",
        hasSubsidy: true | false,
        subsidyInfo: { ... },
        description: "説明"
    },
    // ...
];
```

### データファイルの利点

- ✅ **都道府県ごとに独立**: 他の都道府県に影響しない
- ✅ **更新が簡単**: 該当ファイルだけ編集
- ✅ **並行作業が可能**: 複数人で異なる都道府県を編集可能
- ✅ **バージョン管理が容易**: Git等で差分管理しやすい

## 📢 広告枠について

サイトの運営コストをカバーするため、各ページに広告枠を設置しています。

### 広告枠の設置場所

#### 全国トップページ (`index.html`)
1. **ヘッダー直下**: 728×90 ビッグバナー（レスポンシブ対応）
2. **都道府県選択後**: 336×280 レクタングル大
3. **特徴セクション前**: 300×600 ハーフページ広告
4. **コンテンツ終盤**: 728×90 ビッグバナー

#### 長野県トップページ (`nagano/index.html`)
1. **ヒーロー直下**: 728×90 ビッグバナー（レスポンシブ対応）
2. **プレスリリース後**: 300×250 レクタングル中
3. **特徴セクション前**: 728×90 ビッグバナー

#### 自治体一覧ページ (`nagano/municipalities.html`)
1. **ページヘッダー直下**: 728×90 ビッグバナー
2. **サイドバー**: 300×250 レクタングル中 × 2枠
3. **コンテンツ終盤**: 728×90 ビッグバナー

### 広告枠の仕様

| 広告タイプ | サイズ | クラス名 | 配置場所 |
|----------|-------|---------|----------|
| ビッグバナー | 728×90 | `.ad-header`<br>`.ad-footer` | ヘッダー・フッター付近 |
| レクタングル中 | 300×250 | `.ad-rectangle`<br>`.ad-sidebar` | コンテンツ間・サイドバー |
| レクタングル大 | 336×280 | `.ad-rectangle` | コンテンツ間 |
| ハーフページ | 300×600 | `.ad-halfpage` | サイドバー風レイアウト |

### 広告タグの設置方法

各広告枠には以下のようなHTMLプレースホルダーが設置されています：

**⚠️ 現在、すべての広告枠はコメントアウトされています（非表示）**

広告を表示するには、以下の手順で有効化してください：

#### 広告枠を有効化する手順

1. **該当ページのHTMLファイルを開く**
   - 全国トップ: `index.html`
   - 長野トップ: `nagano/index.html`
   - 自治体一覧: `nagano/municipalities.html`

2. **広告枠のコメントを外す**
   ```html
   <!-- 使用時はコメントを外してください
   <div class="ad-container ad-header">
       <p class="ad-label">Advertisement</p>
       <div class="ad-placeholder">728 × 90 広告枠</div>
   </div>
   -->
   ```
   ↓ コメント部分を削除
   ```html
   <div class="ad-container ad-header">
       <p class="ad-label">Advertisement</p>
       <div class="ad-placeholder">728 × 90 広告枠</div>
   </div>
   ```

3. **広告タグを挿入**（プレースホルダーを置き換え）

```html
<div class="ad-container ad-header">
    <p class="ad-label">Advertisement</p>
    <!-- ここに広告タグを挿入 -->
    <div class="ad-placeholder">728 × 90 広告枠</div>
</div>
```

広告タグの挿入手順：

1. **Google AdSense の場合**
   ```html
   <div class="ad-container ad-header">
       <p class="ad-label">Advertisement</p>
       <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script>
       <ins class="adsbygoogle"
            style="display:block"
            data-ad-client="ca-pub-XXXXXXXXXXXXXXXX"
            data-ad-slot="1234567890"
            data-ad-format="auto"></ins>
       <script>(adsbygoogle = window.adsbygoogle || []).push({});</script>
   </div>
   ```

2. **A8.net などのASP広告の場合**
   ```html
   <div class="ad-container ad-header">
       <p class="ad-label">Advertisement</p>
       <!-- ASPから取得した広告タグを貼り付け -->
       <a href="広告リンク"><img src="バナー画像URL"></a>
   </div>
   ```

3. **プレースホルダーを削除**
   ```html
   <!-- この行を削除 -->
   <div class="ad-placeholder">728 × 90 広告枠</div>
   ```

### レスポンシブ対応

広告枠はCSSメディアクエリでレスポンシブ対応済み：

- **デスクトップ (968px以上)**: すべての広告枠表示
- **タブレット (768px〜968px)**: サイドバー広告を下部に移動
- **モバイル (480px〜768px)**: 広告サイズを調整、縦積み表示
- **極小画面 (480px以下)**: 広告を最小サイズに縮小

### 注意事項

- **プレースホルダー**: 現在はすべての広告枠がコメントアウトされており、非表示です
- **有効化**: 広告を表示する際は、HTMLコメント `<!-- -->` を削除してください
- **段階的導入**: 必要な広告枠だけを選択的に有効化できます
- **広告審査**: 実際の広告タグ挿入前にGoogle AdSense等の審査が必要です
- **掲載ポリシー**: 各広告ネットワークのポリシーに準拠してください
- **パフォーマンス**: 広告スクリプトの非同期読み込み（`async`）を推奨します

### 広告収益の見込み

アクセス数は多くない想定ですが、以下の施策で収益化を図ります：

- ✅ **適切な広告配置**: ユーザー体験を損なわない位置に配置
- ✅ **複数の広告枠**: 各ページに複数の広告枠を設置
- ✅ **レスポンシブ広告**: モバイルユーザーにも最適化
- ✅ **SEO対策**: 検索エンジンからの流入を増やす
- ✅ **コンテンツ充実**: プレスリリース・ブログで再訪を促す

   ```javascript
   // js/main-japan.js
   { 
       name: "北海道", 
       status: "available", 
       url: "hokkaido/index.html",
       municipalities: 45,
       subsidies: 3
   }
   ```

5. **動作確認**
   - ブラウザで表示確認

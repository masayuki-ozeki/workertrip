## 🚀 自動生成ツールの使い方（超簡単）

### 方法A: CSV版（推奨）

```bash
# 1. サンプルCSV生成
cd tools
python generate_facilities.py --sample

# 2. Excelで編集
# facilities_data_sample.csv を開いて施設情報を入力

# 3. 自動生成 + 既存ファイルに統合
python generate_facilities.py facilities_data_sample.csv --merge

# 完了！
```

### 方法B: Googleスプレッドシート版（チーム向け）

```bash
# 1. Googleスプレッドシートを作成（テンプレートをコピー）
# 2. 「リンクを知っている全員が閲覧可能」に設定
# 3. スクリプト実行

python generate_from_sheets.py [スプレッドシートURL]

# 完了！
```

## 🔧 技術的ハイライト

### 実装した工夫

1. **動的レンダリング**
   - JavaScriptで施設カードを自動生成
   - データと表示ロジックの分離

2. **エラーハンドリング**
   - 画像読み込み失敗時のフォールバック
   - リンクがない場合の代替表示

3. **レスポンシブデザイン**
   - デスクトップ: 3カラム
   - タブレット: 2カラム（自動調整）
   - モバイル: 1カラム

4. **SEO対策**
   - `rel="nofollow"` でアフィリエイトリンクマーク
   - 構造化されたHTMLマークアップ

5. **パフォーマンス**
   - 画像は外部CDN（Unsplash）を利用
   - 遅延読み込み対応の準備済み
